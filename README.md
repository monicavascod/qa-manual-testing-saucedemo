# Proyecto de Pruebas Manuales - SauceDemo

## Descripción del proyecto

Este proyecto presenta una evaluación de calidad realizada sobre SauceDemo, una aplicación web que simula el funcionamiento de un comercio electrónico. Durante el proyecto se planifican, diseñan y ejecutan pruebas manuales con el objetivo de evaluar el comportamiento de la aplicación, documentar los defectos encontrados y analizar los riesgos asociados al producto.

## Objetivo

Evaluar la calidad funcional de SauceDemo mediante pruebas manuales, documentando los resultados obtenidos para apoyar la toma de decisiones sobre el estado del producto.

## Aplicación bajo prueba

- **Nombre:** SauceDemo
- **Tipo:** Aplicación web que simula un e-commerce
- **URL:** https://www.saucedemo.com/

## Alcance

Las pruebas se enfocarán en las funcionalidades principales de la aplicación:

- Inicio de sesión
- Catálogo de productos
- Ordenamiento de productos
- Carrito de compras
- Proceso de Checkout
- Confirmación de compra
- Cierre de sesión

## Herramientas utilizadas

- Navegador Google Chrome
- GitHub
- Markdown
- Excel (Casos de prueba)
- Capturas de pantalla

## Entregables

- Análisis del producto
- Plan de pruebas
- Casos de prueba
- Evidencias
- Reportes de bugs
- Informe final de pruebas
